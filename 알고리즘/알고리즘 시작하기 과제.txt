3. 정렬 방법에는 다양한 알고리즘이 있다.

3-1. 최대한 많은 알고리즘에 대해 조사하고 각 알고리즘의 시간 성능을 Big-O notation 을 이용하여 표시하라.
( O 표시는 생략하였습니다 )
정렬 알고리즘 : 데이터를 정렬하는데 사용되는 알고리즘
버블 선택 삽입 : n^2 , 병합 : nlog n

탐색 알고리즘 : 데이터 집합에서 원하는 항목을 찾는데 사용되는 알고리즘
이진 탐색 : log n , 선형 탐색 : n , 해시 탐색 : 1 ~ n

그래프 알고리즘 : 그래프에서 노드와 엣지를 다루는데 사용하는 알고리즘

탐욕 알고리즘 : 최선의 상황을 고르는 알고리즘
n log n , n

문자열 알고리즘 : 문자열  처리를 해결하는 알고리즘
패턴의 길이 + 문자열의 길이

머신러닝 알고리즘 : 데이터로부터 학습 , 예측하는데에 사용되는 알고리즘
일반적으로 계산 불가능
