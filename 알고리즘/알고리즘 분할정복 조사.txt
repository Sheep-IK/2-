Merge sort : n개의 자료가 있으면 n/2개로 그룹을 나누어 하나씩 비교하여 작은 값을 
새로운 list에 넣는다. 이 작업을 계속해서 반복하여 새로운 list에 다 정렬한 다음
새로운 list를 원래 있던 list로 옮기게 되면 정렬이 완료된다.

n개의 자료가 정렬될 때 log n번 돌아가기 때문에 O(n log n) 이 된다

코드 짜기 :
new 배열 전역 선언

merge 함수 {
 각종 선언
 반복문 ( 반으로 나누어서 작은 순서대로 새 배열에 넣기) {
	a b 중에 a가 작으면 a를 배열에
	               b가 작으면 b를 배열에
	그 다음 그룹
 나머지 넣어주기	
 } //반복문 끝

정렬된 배열을 원래 배열로 이동(반복)
}


Quick sort : list 중에 하나를 선택하고(피벗) 피벗보다 작은 자료들은 피벗기준 왼쪽,
보다 큰 자료들은 피벗기준 오른쪽으로 이동한다.
왼쪽에서 새로운 피벗을 정해서 왼쪽 오른쪽으로 나누는 정렬을 반복한다.
더이상 나눌 수 없으면 끝낸다.

코드 짜기 :
배열
피봇 선언 ( 배열 전체 / 2 번째 값)
반복문 ( 맨 앞이 끝이랑 같거나 작을때)
       반복문 ( 피벗 왼쪽 값이 피벗보다 작을때 )
	다음 왼쪽 값
       반복문 ( 피벗 오른쪽 값이 피벗보다 클때 )
	다음 오른쪽 값
       오른쪽값이 왼쪽 값보다 작으면 서로 바꾸기
 
정렬 완료
